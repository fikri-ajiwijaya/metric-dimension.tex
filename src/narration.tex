\section{Introduction}

The theoretical part of this paper is divided into three major parts. The first of which consists of the heavy lifting of formally defining notations for slicing, dicing, and filtering of matrices, tuples, and sets. While it may seem trivial and verbose at first glance, it is of utmost importance due to the lack of standard notations for these basic operations. Part two and part three extensively relies on using these carefully defined operations, without which would be incomprehensible to read. This rigor is both necessary and unnegotiable. Part two consists of a quick recall on the metric dimension problem definition, then followed by the first major contribution of this paper; that is, the journey in perspective shift of metric dimension reformulated in terms of slicing, dicing, and filtering. After all this gruelling groundwork, we are finally able to introduce the theoretical part of our result in part three, where we formulate constraints for metric dimension in satisfiability modulo theories, through careful solution space analysis.

\section{Preliminaries}

Let \(G\) be a connected graph with \(n\) vertices \(V = \{v, \cdots\}\) and \(m\) edges \(E = \{e, \cdots\}\). There exists a function \(d(u, v)\), for any \(u, v \in V\), that gives the distance from \(u\) to \(v\). Then, for any \(u, v, w \in V\) and \(e \in E\), if \(e\) is the edge that connects \(v\) and \(w\), it is defined \(d(u, e) = \mathrm{min}(d(u, v), d(u, w))\). For the rest of this paper, let us take a subset of \(k\) vertices \(W = \{w, \cdots\} \subseteq V\).

The representation of \(v \in V\) with respect to \(W\) is defined as the \(k\)-tuple \(r(v|W) = (\ d(v, w),\ \cdots\ )\), and the edge representation of \(e \in E\) with respect to \(W\) is defined as the \(k\)-tuple \(r(e|W) = (\ d(e, w),\ \cdots\ )\). \(W\) is said to be a resolving set when all representations with respect to it are unique, and \(W\) is said to be an edge resolving set when all edge representations with respect to it are unique.
\[
	W \text{ is a resolving set} \iff \forall u, v \in V, u \neq v,\enspace r(u|W) \neq r(v|W)
\]
\[
	W \text{ is an edge resolving set} \iff \forall e, f \in E, e \neq f,\enspace r(e|W) \neq r(f|W)
\]
Conversely, \(W\) is said to be a non-resolving set when there exists a pair of non-unique representations, and \(W\) is said to be an edge non-resolving set when there exists a pair of non-unique edge representations.
\[
	W \text{ is a non-resolving set} \iff \exists u, v \in V, u \neq v,\enspace r(u|W) = r(v|W)
\]
\[
	W \text{ is an edge non-resolving set} \iff \exists e, f \in E, e \neq f,\enspace r(e|W) = r(f|W)
\]
The power set \(\mathcal{P}(V)\) is therefore partitioned into the set of resolving sets and the set of non-resolving sets. Likewise, it is also partitioned into the set of edge resolving sets and the set of edge non-resolving sets.

Both resolving sets and edge resolving sets has the property of upward monotonicity.
\[
	W \text{ is a resolving set},\enspace W \subseteq Z \implies Z \text{ is a resolving set}
\]
\[
	W \text{ is an edge resolving set},\enspace W \subseteq Z \implies Z \text{ is an edge resolving set}
\]
While both non-resolving sets and edge resolving sets has the property of downward monotonicity.
\[
	W \text{ is a non-resolving set},\enspace Z \subseteq W \implies Z \text{ is a non-resolving set}
\]
\[
	W \text{ is an edge non-resolving set},\enspace Z \subseteq W \implies Z \text{ is an edge non-resolving set}
\]

Ultimately, the metric dimension of \(G\) is the minimum cardinality of its resolving sets, and the edge metric dimension of \(G\) is the minimum cardinality of its edge resolving sets.

\section{Slicing, Dicing, and Filtering}

For any set \(U\) with \(k\) elements, we denote its cardinality by \(|U| = k\), and its power set by \(\mathcal{P}(U)\). For any subset \(R \subseteq U\), we denote its complement by \(\widetilde{R} = U \backslash R\).

For any \(k\)-tuple \(C\), we denote its length by \(|C| = k\), and its element at the \(i\)-th position by \(C(i)\). Let us use \(c \in C\) to denote that \(c\) is an element in \(C\), and \(\mathrm{idx}(c)\) to denote the element index of \(c\) in \(C\).

A correspondence is said to exists between a set \(U\) and a \(k\)-tuple \(C\) if and only if there exists a bijective function \(\mathrm{idx} : U \to \{1, \cdots, |C|\}\). For any element \(u \in U\), let us define selecting \(C\) by \(u\) be an element in \(C\) as \(C(u) = C(\mathrm{idx}(u))\). For any subset \(U' = \{u, \cdots\} \subseteq U\), let us define filtering \(C\) by \(U'\) (preserving element order in \(C\)) to be a \(|U'|\)-subtuple as \(C(U') = (\ C(u),\ \cdots\ )\).

Let us define some element-wise operations on \(k\)-tuples; the element-wise negation, \(\neg C = (\ \neg C(i),\ \cdots\ )\); the element-wise disjunction, \(C_1 \vee C_2 = (\ C_1(i) \vee C_2(i),\ \cdots\ )\); and the element-wise equality, \(C_1\ \ddot{=}\ C_2 = (\ C_1(i) = C_2(i),\ \cdots\ )\). Let us also define \((\mathrm{True})^k\) as the Boolean \(k\)-tuple consisting of all \(\mathrm{True}\) values.

For any Boolean \(k\)-tuple \(B\) and any set \(U\), where there exists a correspondence between them, let us define filtering \(U\) by \(B\) as \(U(B) = \{u \in U : B(u) = \mathrm{True}\}\). And for any integer \(k\)-tuple \(I\), where there exists a correspondence between it and the set \(U\), let us define filtering \(U\) by \(I\) as \(U(I) = \{u \in U : I(u) \neq 0\}\). Then, let us extend this to dicing, that for any set of \(k\)-tuples \(\mathcal{C} = \{C, \cdots\}\), where there exists a correspondence between \(U\) and each \(k\)-tuple in \(\mathcal{C}\), let us define dicing \(U\) by \(\mathcal{C}\) as \(U(\mathcal{C}) = \{\ U(C),\ \cdots\ \}\).

For any \(n \times m\) matrix \(A\), we denote the number of rows in \(A\) by \(|A|_{\mathrm{row}} = n\), the number of columns in \(A\) by \(|A|_{\mathrm{col}} = m\), and its element at the \(i\)-th row and \(j\)-th column by \(A(i, j)\). Let us use \(\mathcal{R} \overset{\mathrm{row}}{\in} A\) to denote that \(\mathcal{R}\) is a row in \(A\) as an \(m\)-tuple, \(\mathcal{K} \overset{\mathrm{col}}{\in} A\) to denote that \(\mathcal{K}\) is a column in \(A\) as an \(n\)-tuple, \(\mathrm{idx}(\mathcal{R})\) to denote the row index of \(\mathcal{R}\) in \(A\), and \(\mathrm{idx}(\mathcal{K})\) to denote the column index of \(\mathcal{K}\) in \(A\). Let us define \(A(i, \cdot)\) to be the slicing the \(i\)-th row of \(A\) as an \(m\)-tuple, and \(A(\cdot, j)\) to be slicing the \(j\)-th column of \(A\) as an \(n\)-tuple.

A row correspondence is said to exists between a set \(S\) and an \(n \times m\) matrix \(A\) if and only if there exists a bijective function \(\mathrm{idx} : S \to \{1, \cdots, |A|_{\mathrm{row}}\}\), and a column correspondence is said to exists between a set \(T\) and a matrix \(A\) if and only if there exists a bijective function \(\mathrm{idx} : T \to \{1, \cdots, |A|_{\mathrm{col}}\}\). For any element \(s \in S\), let us define row slicing \(A\) by \(s\) to be an \(m\)-tuple as \(A(s, \cdot) = A(\mathrm{idx}(s), \cdot)\), and, for any element \(t \in T\), let us define column slicing \(A\) by \(t\) to be an \(n\)-tuple as \(A(\cdot, t) = A(\cdot, \mathrm{idx}(t))\). Then, for any subset \(S' = \{s, \cdots\} \subseteq S\), let us define row filtering \(A\) by \(S'\) (preserving row order in \(A\)) to be an \(|S'| \times m\) submatrix as
\[
	A(S', \cdot) = \left[
	\begin{array}{c}
		A(s, \cdot) \\
		\vdots
	\end{array}
	\right] ,
\]
and, for any subset \(T' = \{t, \cdots\} \subseteq T\), let us define column filtering \(A\) by \(T'\) (preserving column order in \(A\)) to be an \(n \times |T'|\) submatrix as \(A(\cdot, T) = \left[\ A(\cdot, t)\ \cdots\ \right]\).

Finally, let us assert some matrix element selection, subcolumn slicing, subrow slicing, and submatrix dicing identities. They are,
\[
\begin{array}{l@{\ }l@{\ }ll}
	A(s, t) &= A(s, \cdot)(t) &= A(\cdot, t)(s) &, \\
	A(S', t) &= A(S', \cdot)(\cdot, t) &= A(\cdot, t)(S') &, \\
	A(s, T') &= A(s, \cdot)(T') &= A(\cdot, T')(s, \cdot) &, \text{ and} \\
	A(S', T') &= A(S', \cdot)(\cdot, T') &= A(\cdot, T')(S', \cdot) &.
\end{array}
\]

\section{Distance Matrix}

Observe that \(V\) has correspondence with both \(r(v|V)\) and \(r(e|V)\). This meas we may rewrite the representation definition with respect to \(W\) to filter from the representation with respect to \(V\), that is \(r(v|W) = r(v|V)(W)\). Likewise, we are able to rewrite the edge representation definition in terms of tuple filtering as \(r(e|W) = r(e|V)(W)\).

Let's now shift our focus a little bit. Let us define a new structure called the representation matrix with respect to \(W\) as \(r(V|W) = \left[\ r(v|W)\ \cdots\ \right]\). We are now able to rewrite the representation definition in terms of the representation matrix by column slicing, that is \(r(v|W) = r(V|W)(\cdot, v)\). Likewise, by defining the edge representation matrix \(r(E|W) = \left[\ r(e|W)\ \cdots\ \right]\), we can rewrite the edge representation definition to be \(r(e|W) = r(E|W)(\cdot, e)\).

Let us rewrite the resolving set and edge resolving set criteria in terms of this new structure.
\[
	W \text{ is a resolving set} \iff \forall u, v \in V, u \neq v,\enspace r(V|W)(\cdot, u) \neq r(V|W)(\cdot, v)
\]
\[
	W \text{ is an edge resolving set} \iff \forall e, f \in E, e \neq f,\enspace r(E|W)(\cdot, e) \neq r(E|W)(\cdot, f)
\]
By careful examination, these criteria essentially now says that determining \(W\) to be a resolving set is a matter of showing that no two columns in the representation matrix with respect to \(W\) are the same, and, by the same reasoning, determining \(W\) to be an edge resolving set has become a matter of showing that all columns of the edge representation matrix are unique. Using column notations directly, this may be expressed better as follow:
\[
	W \text{ is a resolving set} \iff \forall C, \mathcal{C} \overset{\mathrm{col}}{\in} r(V|W),\ \mathrm{idx}(C) \neq \mathrm{idx}(\mathcal{C}),\ C \neq \mathcal{C}
\]
\[
	W \text{ is an edge resolving set} \iff \forall C, \mathcal{C} \overset{\mathrm{col}}{\in} r(E|W),\ \mathrm{idx}(C) \neq \mathrm{idx}(\mathcal{C}),\ C \neq \mathcal{C}
\]

Notice that, because \(r(v|V) = r(V|V)(\cdot, v)\), we obtain \(r(v|W) = r(V|V)(\cdot, v)(W)\), and similarly, we also obtain \(r(e|W) = r(E|V)(\cdot, e)(W)\). Using the subcolumn slicing identity, we then have \(r(v|W) = r(V|V)(W, \cdot)(\cdot, v)\) and \(r(e|W) = r(E|V)(W, \cdot)(\cdot, e)\). By forgoing the final column slicing, we have an alternative way to attain the representation matrix with respect to \(W\) as a submatrix of \(r(V|V)\) by row filtering, that is \(r(V|W) = r(V|V)(W, \cdot)\). Likewise, we have the alternative edge representation submatrix form as \(r(E|W) = r(E|V)(W, \cdot)\).

The matrices \(r(V|V)\) and \(r(E|V)\) are more commonly known as the distance matrix and the edge distance matrix of \(G\), where by we can obtain any distance in \(G\) from either of these matrices.
\[
	u, v \in V,\enspace d(u, v) = r(V|V)(\cdot, u)(v)
\]
\[
	v \in V, e \in E, \enspace d(v, e) = r(E|V)(\cdot, v)(e)
\]
