	\section{Preliminaries}

	\subsection{Notations on tuples}

	Let \(c_1, c_2, \cdots, c_k\) be constants. They may be arranged into a \(k\)-tuple as follow: \((c_1, c_2, \cdots, c_k)\).

	Let \(C\) be a \(k\)-tuple. \(C(i)\) is the \(i\)-th element in tuple \(C\).

	Let \(U\) be a set where \(|U| = k\). For \(U\) to have correspondence with \(C\) means there exists a bijective function \({\mathtt{idx} : U \to \{1, \cdots, k\}}\).

	If \(u \in U\), then \(C(u) = C(\mathtt{idx}(u))\). If \(\{u_1, \cdots, u_r\} \subseteq U\), then, preserving the element order of \(C\),
	\[
		C(\{u_1,\cdots, u_r\}) = (C(u_1), \cdots, C(u_r))
	\]

	Let \(C_1\) and \(C_2\) be \(k\)-tuples.
	\[
		(C_1 = C_2) = (C_1(1) = C_2(1), \cdots, C_1(k) = C_2(k))
	\]

	\subsection{Notations on matrices}

	Let \(a_1, a_2, \cdots, a_n\) be \(m\)-tuples. They may be arranged into the columns of an \(m \times n\) matrix as follow: \(\left[ a_1\enspace a_2\enspace \cdots\enspace a_n \right]\).

	Let \(A\) be an \(n \times m\) matrix. \(A(i, j)\) is the element at the \(i\)-th row and \(j\)-th column in matrix \(A\).
	\[ A(i, \star) = (A(i, 1), \cdots, A(i, m)) \]
	\[ A(\star, j) = (A(1, j), \cdots, A(n, j)) \]

	Let \(S\) be a set where \(|S| = n\). For \(S\) to have row correspondence with \(A\) means there exists a bijective function \({\mathtt{idx} : S \to \{1, \cdots, n\}}\).

	If \(s \in S\), then \(A(s, \star) = A(\mathtt{idx}(s), \star)\). If \(\{s_1, \cdots, s_r\} \subseteq S\), then, preserving the row order of \(A\),
	\[
		A(\{s_1, \cdots, s_r\}, \star) = \left[
		\begin{array}{c}
			A(s_1, \star) \\
			\vdots \\
			A(s_r, \star)
		\end{array}
		\right]
	\]

	Let \(T\) be a set where \(|T| = m\). For \(T\) to have column correspondence with \(A\) means there exists a bijective function \({\mathtt{idx} : T \to \{1, \cdots, m\}}\).

	If \(t \in T\), then \(A(\star, t) = A(\star, \mathtt{idx}(t))\).

	\subsection{Notations on graphs}

	\subsubsection{Distance}

	Let \(G\) be a graph. Let \(V\) be the set of vertices in \(G\). Let \(E\) be the set of edges in \(G\).

	For \(u, v \in V\), the distance from \(u\) to \(v\) is denoted by \(d(u, v)\).

	For \(u, v, w \in V\) and \(e \in E\), if \(e\) is an edge between \(v\) and \(w\), then the distance from \(u\) to \(e\) is
	\[
		d(u, e) = \mathrm{min}(d(u, v), d(u, w))
	\]

	\subsubsection{Representation}

	For \(v \in V\) and \(W = \{w_1, w_2, \cdots, w_k\} \subseteq V\), the representation of \(v\) with respect to \(W\) is
	\[
		r(v|W) = (d(w_1, v), d(w_2, v), \cdots, d(w_k, v))
	\]

	For \(e \in E\) and \(W = \{w_1, w_2, \cdots, w_k\} \subseteq V\), the representation of \(e\) with respect to \(W\) is
	\[
		r(e|W) = (d(w_1, e), d(w_2, e), \cdots, d(w_k, e))
	\]

	\subsubsection{Resolving set}

	For \(W\) to be a resolving set means that \(\forall u, v \in V, u \neq v \implies r(u|W) \neq r(v|W)\).

	For \(W\) to be an edge resolving set means that \(\forall e, f \in E, e \neq f \implies r(e|W) \neq r(f|W)\).

	\subsubsection{Non-resolving set}

	If \(W\) is not a resolving set, then \(W\) is a non-resolving set. \(\mathcal{P}(V)\) is partitioned into the set of resolving sets and the set of non-resolving sets.

	If \(W\) is not an edge resolving set, then \(W\) is an edge non-resolving set. \(\mathcal{P}(V)\) is partitioned into the set of edge resolving sets and the set of edge non-resolving sets.

	\subsubsection{Monotonicity}

	If \(W\) is a resolving set, then a superset of \(W\) is a resolving set. If \(W\) is a non-resolving set, then a subset of \(W\) is a non-resolving set.

	If \(W\) is an edge resolving set, then a superset of \(W\) is an edge resolving set. If \(W\) is an edge non-resolving set, then a subset of \(W\) is an edge non-resolving set.

	\subsubsection{Metric dimension}

	The metric dimension of a graph is the minimum cardinality of resolving sets of the graph.

	The edge metric dimension of a graph is the minimum cardinality of edge resolving sets of the graph.

	\section{Distance Matrix}

	Suppose \(G\) is a graph with a set of vertices \(V = \{v_1, v_2, \cdots, v_n\}\) and a set of edges \(E = \{e_1, e_2, \cdots, e_m\}\).

	Let us compute for a vertex \(v \in V\) its representation with respect to \(V\). We would obtain the tuple \(r(v|V) = (d(v_1, v), d(v_2, v), \cdots, d(v_n, v))\). Observe that, this tuple contains the distance from every other vertices to \(v\), where \(\forall u \in V, d(u, v) = r(v|V)(u)\). Such, for any \(W \subseteq V\), the representation of \(v\) with respect to \(W\) may be acquired by slicing directly from this tuple; that is, \(r(v|W) = r(v|V)(W)\).

	Next, let us revisit the notion of resolving set. The set \(W \subseteq V\) is a resolving set of \(G\) if and only if the representation of all vertices with respect to it are unique. Thus, by definition, we must compute all possible representations with respect to \(W\) to even be able to begin quantifying whether \(W\) fulfill this criteria.

	However, since determining the metric dimension of \(G\) is fundamentally a process of minimizing the cardinality of the resolving set. As such, by virtue of the search process, we'd have to evaluate various potential \(W \subseteq V\). Therefore, it'd be best to reuse as much computation as possible. This, we can do by computing all possible representations with respect to \(V\) instead, once, at the beginning. Then, all further possible representations may be obtained by slicing from these collections of precomputed values.

	Fortunately, there exists already a structure that contains all this information. It is called the distance matrix, defined as follow:
	\[
		D^{(V)} = \left[
		\begin{array}{cccc}
			d(v_1, v_1) & d(v_1, v_2) & \cdots & d(v_1, v_n) \\
			d(v_2, v_1) & d(v_2, v_2) & \cdots & d(v_2, v_n) \\
			\vdots & \vdots & \ddots & \vdots \\
			d(v_n, v_1) & d(v_n, v_2) & \cdots & d(v_n, v_n)
		\end{array}
		\right]
	\]
	This distance matrix has dimension \(n \times n\). The construction is made specifically such that \(V\) has both row and column correspondence with \(D^{(V)}\). Remark that the columns of the distance matrix are exactly the representations with respect to \(V\). Through slicing, we have \(r(v|V) = D^{(V)}(\star, v)\) for any \(v \in V\). Hence, together with our previous observation, we are able to obtain any representation from the distance matrix using \(r(v|W) = D^{(V)}(\star, v)(W)\).

	Equivalently, we can first select the rows of \(D^{(V)}\) according to \(W\). The columns of the submatrix \(D^{(V)}(W, \star)\) are exactly the representations with respect to \(W\). Then, we can select a column from the submatrix to obtain the representation for a specific vertex, giving \(r(v|W) = D^{(V)}(W, \star)(v)\).

	If \(E = \{e_1, e_2, \cdots, e_m\}\), then the edge distance matrix of \(G\) has dimension \(n \times m\), and is as follow:
	\[
		D^{(E)} = \left[
		\begin{array}{cccc}
			d(v_1, e_1) & d(v_1, e_2) & \cdots & d(v_1, e_m) \\
			d(v_2, e_1) & d(v_2, e_2) & \cdots & d(v_2, e_m) \\
			\vdots & \vdots & \ddots & \vdots \\
			d(v_n, e_1) & d(v_n, e_2) & \cdots & d(v_n, e_m)
		\end{array}
		\right]
	\]
	\(V\) has row correspondence and \(E\) has column correspondence with \(D^{(E)}\).

	Notice that the columns of \(D^{(E)}\) are representations with respect to \(V\); that is, for \(e \in E\), \(r(e|V) = D^{(E)}(\star, e)\).

	Thus, the representation with respect to \(W\) is \(r(e|W) = D^{(E)}(\star, e)(W)\).

	\section{Distance Similarity Matrix}

	A pair of \(k\)-tuples \(c\) and \(d\) is said to be equal if and only if \(\forall i \in \{1, \cdots, k\}, c(i) = d(i)\). This process, however, may be separated into two steps. One is to perform the pair-wise element comparisons to return Boolean values and keep their arranngement as a resulting \(k\)-Boolean tuple; in other words, \((c = d)\). Then, \(c = d \iff \forall i, (c = d)(i) = \mathtt{True}\). Conversely, \(c \neq d \iff \exists i, (c = d)(i) = \mathtt{False}\).

	This two step evaluation is useful within the context of representations comparisons. For \(u, v \in V\), we may evaluate the tuple \((r(u|V) = r(v|V))\) once. Then, for any \(W \subseteq V\), evaluating whether the representations of \(u\) and \(v\) with respect to \(W\) differs or equal may be taken from slicing the previously computed tuple: \((r(u|V) = r(v|V))(W)\).

	For \(u, v \in V, u \neq v\) and \(W \subseteq V\), whether the representation of \(u\) and \(v\) with respect to \(W\) differs, it may be tested through the following criteria:
		\[ r(u|W) \neq r(v|W) \iff \mathtt{False} \in (r(u|W) = r(v|W)) \]

	We may evaluate \((r(u|V) = r(v|V))\) once and realize that \(V\) has a correspondence with the resulting tuple. As such, the test for arbitrary \(W\) may be adjusted to reuse computation as follow:
		\[ r(u|W) \neq r(v|W) \iff \mathtt{False} \in (r(u|V) = r(v|V))(W) \]

	To determine whether \(W\) is a resolving set or a non-resolving set, it is necessary to perform this test for all combination of pairs of vertices in \(V\). We may evaluate all of these possible combinations with respect to \(V\) once and arrange them into columns in a matrix as follow:
	\[
		B^{(V)} = \left[\;
			(r(v_1|V)\!=\!r(v_2|V))\;
			\cdots\;
			(r(v_1|V)\!=\!r(v_n|V))
			\cdots\;
			(r(v_{n-1}|V)\!=\!r(v_n|V))\;
		\right]
	\]
	or by reusing \(D^{(V)}\) computation
	\[
		B^{(V)} = \left[\;
			(D^{(V)}(\star, v_1)\!=\!D^{(V)}(\star, v_2))\;
			\cdots\;
			(D^{(V)}(\star, v_1)\!=\!D^{(V)}(\star, v_n))
			\cdots\;
			(D^{(V)}(\star, v_{n-1})\!=\!D^{(V)}(\star, v_n))\;
		\right]
	\]
	This matrix is called the distance similarity matrix of \(G\), and has a dimension of \(n \times \binom{n}{2}\). Here, \(B^{(V)}\) preserves its row correspondence with \(V\). By observing whether all columns in \(B^{(V)}(W)\) contains \(\mathtt{False}\), \(W\) can be concluded to be a resolving set; or more rigorously,
	\[
		W \text{ is a resolving set} \iff \forall j, \enspace \texttt{False} \in B^{(V)}(W, \star)(\star, j)
	\]

	Similarly, we may construct an edge distance similarity matrix of dimension \(n \times \binom{m}{2}\) as follow:
	\[
		B^{(E)} = \left[\;
			(r(e_1|V)\!=\!r(e_2|V))\;
			\cdots\;
			(r(e_1|V)\!=\!r(e_m|V))
			\cdots\;
			(r(e_{m-1}|V)\!=\!r(e_m|V))\;
		\right]
	\]
	or by reusing \(D^{(E)}\) computation
	\[
		B^{(E)} = \left[\;
			(D^{(E)}(\star, e_1)\!=\!D^{(E)}(\star, e_2))\;
			\cdots\;
			(D^{(E)}(\star, e_1)\!=\!D^{(E)}(\star, e_m))
			\cdots\;
			(D^{(E)}(\star, e_{m-1})\!=\!D^{(E)}(\star, e_m))\;
		\right]
	\]
	Likewise, \(B^{(E)}\) preserves its row correspondence with \(V\). By observing whether all columns in \(B^{(E)}(W)\) contains \(\mathtt{False}\), \(W\) can be concluded to be an edge resolving set; or more rigorously,
	\[
		W \text{ is an edge resolving set} \iff \forall j, \enspace \texttt{False} \in B^{(E)}(W, \star)(\star, j)
	\]
